## 开发环境

- [tauri 开发环境](https://v2.tauri.app/start/prerequisites/)
- rust
- nodejs & pnpm

```bash
pnpm i
pnpm tauri dev
```

## 食用方法

将您的 `cses.json` 文件放置到 `$XDG_CONFIG_HOME/class-table/cses.json`
